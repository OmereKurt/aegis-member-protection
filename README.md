
## Demo Flow

A simple demo flow for Analyst Copilot:

1. Open the dashboard homepage
2. Review total case counts and severity breakdown
3. Submit a suspicious login alert
4. Submit a phishing email alert
5. Open the saved cases page
6. Filter/search for a specific case
7. Open the case detail page
8. Review normalized alert data, reasons, enrichment, and recommended actions
