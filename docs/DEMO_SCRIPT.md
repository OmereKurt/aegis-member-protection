# Analyst Copilot Demo Script

## Goal
Show how raw alerts become analyst-ready cases.

## Demo Steps

### 1. Open the homepage
- Explain the dashboard
- Point out total cases and severity distribution

### 2. Submit a suspicious login alert
- Mention impossible travel
- Mention failed logins
- Mention risky source IP
- Show returned severity, reasons, and recommendations

### 3. Submit a phishing alert
- Mention suspicious sender domain
- Mention display name mismatch
- Mention URL or attachment presence
- Show returned severity, reasons, and recommendations

### 4. Open saved cases
- Show that cases are stored
- Show search and severity filtering

### 5. Open case detail
- Show normalized alert structure
- Show enrichment
- Show recommended actions

## Key Talking Points
- deterministic scoring instead of black-box decisions
- normalized schema across alert types
- product-style dashboard and case review workflow
- easy path for future expansion into real enrichment and integrations
